ll alias 재정의

December 11 2017

목적 : ll 명령어를 입력시 숨김파일 보여주고 싶을때 ( -a 옵션이 빠져있음)
CentOS7 만 확인함.

/etc/profile.d/colorls.sh 거의 맨아랫줄에

alias ll='ls -l --color=auto' 2>/dev/null 에 있음

root 권한으로 해당 파일
alias ll='ls -al --color=auto' 2>/dev/null 로 수정하고 (a 추가)

source /etc/profile.d/colorls.sh 하면 됨.

또는 ~/.bash_profile 에
alias ll="ls -alGh" 재정의해도 됨
