centos7 root 패스워드 분실

December 23 2017

  1. 증상
    # su root
    비번 입력시 authentication failure 나와서 보니…
    root 로 ssh, 로컬(콘솔) 모두 안됨

  2. 원인
    root 비번이 바뀐 원인은 모르겠음. 다만,
    초기 셋팅시 사용자계정(flapper)을 sudoer 에 추가하지 않았음
    만약에 추가했다면 sudo -i 로 가능했을 수도 (아니다.. 로컬 진입도 안되었는데 ㅠ)
    참고링크(그냥 참고다 혹시 su 안될때 보기 위해서)
    비슷한 다른링크

  3. 해결
    콘솔에서 비번 초기화함
    http://blog.naver.com/PostView.nhn?blogId=dme1004&logNo=220692886434

약간 어려운 부분은
ro => rw, rhgb quiet => init=/bin/bash 로 수정후 Ctrl+X 키를 누르자.`
