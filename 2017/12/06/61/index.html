bash 화면분할 screen 명령어

December 06 2017

편집기를 여러개 띄우면 마우스로 조작해야해서 귀찮고
키보드로 다 할수 있는 멋진 툴,osx 에는 원래 있고 ubuntu 나 centos 는 설치해야함

screen 으로 시작
Ctrl + a S 스크린 창 분할하기
Ctrl + a [tab키] 분할된 스크린창 이동하기
Ctrl + a c 새로운 가상터미널 생성.
Ctrl + a d 현재 상태에서 detach 시킴.
