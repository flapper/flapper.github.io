CentOS7 설치시 Error setting up base repository

August 05 2017

  1. 증상
    SOFTWARE > INSTALLATION SOURCE 항목에서
    Error setting up base repository 보이고 설치 안됨

  2. 원인
    윈도우에서 USB 만들때 LiveUSB creator 는 안됨

  3. 해결
    mac 에서 만들던지.. 또는
    Win32 Disk Manager 써야함
    [참조]{https://serverfault.com/questions/710541/centos-7-1503-getting-error-setting-up-base-repository-during-installation}

파일 첨부 : win32diskimager-1.0.0-install.exe

  • USB 포맷시는 extFAT 로 포맷한다.
  • 설치후 Image File 선택시 iso 인 경우 파일 확장자만 iso 에서 img 로 바꾸면 인식됨
