macos 특수기호 자주 쓸때

August 19 2017

  • 한자
    한글 입력후 option+return

  • 특수기호 : 자주 안쓰면 화면 오른쪽위
    [이모티콘 및 기호보기] 선택해서 쓰면 됨

  • 특수기호 : 자주 쓰면 아래 파일
    Keyboard-en.plist
    여기에 적용
    /System/Library/Input Methods/PressAndHold.app/Contents/Resources/
    In macOS Sierra, this path has changed. The path is now
    /System/Library/Input Methods/PressAndHold.app/Contents/PlugIns/PAH_Extension.appex/Contents/Resources/

    참조링크 : http://macnews.tistory.com/429, https://www.reddit.com/r/MacOS/comments/55hrki/cannot_find_the_keyboardenplist_file_in/
